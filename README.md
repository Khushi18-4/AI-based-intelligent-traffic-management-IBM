# 🚦 AI-Based Smart Traffic Management System/IBM project

## 🌆 Overview
This project uses Machine Learning to predict traffic congestion levels and optimize signal timing for better traffic flow.AI-based traffic management system using ML to predict congestion levels and optimize signal timing. Includes Streamlit dashboard for real-time input and smart traffic control simulation.

## 🎯 Features
- 🚗 Traffic Prediction (Low / Medium / High)
- ⏱ Smart Signal Timing
- 🌦 Weather-based Analysis
- 💻 Interactive Dashboard (Streamlit)

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📊 Inputs
- Vehicle Count
- Speed
- Time of Day
- Weather
- Rush Hour

## 🚀 Output
- Traffic Level
- Signal Time
